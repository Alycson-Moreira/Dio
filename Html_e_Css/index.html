  1. Uso das tags H,Oque são as tags H?

      Os elementos HTML h1 até h6 representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo.

      fonte: mdn web docs

      exemplos:

    • Titulo h1

    • Titulo h2

    • Titulo h3

    • Titulo h4

    • Titulo h5
    • Titulo h6

  2. TAG small

    A tag <small></small> define um texto menor (como direitos autorais e outros comentários secundários).

    exemplo:

    O texto será exibido menor

    O texto será exibido menor

    fonte: w3schools

  3. Abreviações e exclusão de textos

    A tag <abbr> define uma abreviação ou um acrônimo
